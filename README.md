# fetcher
在开发微服务的过程中，后端的API需要不断的测试来保证在开发过程中的鲁棒性，使用curl，webdriver或许是不错的，但是有时候每一两次请求都要重新编写配置文件会比较麻烦，而微服务本身又不会有太多的API暴露出来，因此一个趁手的小工具是非常必要的，fetcher就是这样一个小工具



# 技术栈

1. 适用于Electron的UI布局框架[Photon](https://github.com/connors/photon)
2. AJAX使用的库 [jquery](http://jquery.com/)
3. 表单验证使用的插件[jquery validation](http://www.jqueryvalidation.org/)
